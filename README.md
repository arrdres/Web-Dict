# Web-Dict
Web渗透测试中常用的目录扫描字典

从多个字典项目中整合再去重，旨在提高渗透测试中对网站目录扫描的效率。


#  字典说明
JSP：7万 | ASP：39万 | ASPX：12万 | PHP：42万

常规目录：40万 | 常见目录1：80万 | 常见目录2：43万 | 常见目录3：52万

......

包含常见后台目录、备份文件、上传路径等，完全能够满足渗透测试中扫描目录的需求。
